删除有序链表的重复数字2

发表于 2019-10-26 更新于 2019-10-27 分类于 leetcode

和上一题区别在于，这个题目需要删除所有重复的数字。所以我们考虑在判定指针前面再加一个指针。如1，2，2，3初始时pre指针在1的前面，cur在1处。因为cur的val与cur后继的val不相等，所以，equal等于false，pre变到1，cur变到第一个2处（cur = cur.next）。之后，因为cur的后继2与cur相等，所以equal等于true，cur移到第二个2处。将1的后继变为第二个2的后继即3处,equal变为false，cur指针变到3处（cur=cur.next)

删除有序链表的重复数字1

发表于 2019-10-26 更新于 2019-10-27 分类于 leetcode

将cur指针指在head节点。在cur且cur的后继不为空情况下，判定cur与cur的后继是否相等。如果相等，将cur后继的后继变为cur的后继，如：1->1->2->3 中2变为1的后继，cur还是在1处。若不相等，则cur的指针后移，如cur后移至2处。当cur的后继为空时，如3的后继为空时，返回head。由于jvm有垃圾回收机制，所以节点可以不用回收。

合并两个有序链表

发表于 2019-10-26 更新于 2019-10-27 分类于 leetcode
  • 迭代法

创建一个新链表cur，用h指向cur节点。如果l1的值小于l2的值，则h的后继变为l1节点，同时将h与l1节点的指针后移。反之，则对l2进行同样的操作。如果l1变为null了，直接将l2贴在h后面。反之，对l2进行同样操作。最后，返回cur.next（头节点不存值）
