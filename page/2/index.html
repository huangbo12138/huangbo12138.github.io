删除有序链表的重复数字2

发表于 2019-10-26 更新于 2019-10-27 分类于 leetcode

和上一题区别在于，这个题目需要删除所有重复的数字。所以我们考虑在判定指针前面再加一个指针。如1，2，2，3初始时pre指针在1的前面，cur在1处。因为cur的val与cur后继的val不相等，所以，equal等于false，pre变到1，cur变到第一个2处（cur = cur.next）。之后，因为cur的后继2与cur相等，所以equal等于true，cur移到第二个2处。将1的后继变为第二个2的后继即3处,equal变为false，cur指针变到3处（cur=cur.next)

删除有序链表的重复数字1

发表于 2019-10-26 更新于 2019-10-27 分类于 leetcode

将cur指针指在head节点。在cur且cur的后继不为空情况下，判定cur与cur的后继是否相等。如果相等，将cur后继的后继变为cur的后继，如：1->1->2->3 中2变为1的后继，cur还是在1处。若不相等，则cur的指针后移，如cur后移至2处。当cur的后继为空时，如3的后继为空时，返回head。由于jvm有垃圾回收机制，所以节点可以不用回收。

创建一个新链表cur，用h指向cur节点。如果l1的值小于l2的值，则h的后继变为l1节点，同时将h与l1节点的指针后移。反之，则对l2进行同样的操作。如果l1变为null了，直接将l2贴在h后面。反之，对l2进行同样操作。最后，返回cur.next（头节点不存值）

上述代码中，日志信息与业务逻辑的耦合度很高，不利于系统的维护，使用AOP可以优化。可以通过动态代理来实现AOP

给业务逻辑找一个代理，打印日志信息的工作交给代理来做，这样的话业务代码就只需要关注自身的业务即可。

以上通过动态代理的方法实现AOP的过程，复杂且不好理解。Spring对AOP进行了封装，可以用面向对象的方法来实现AOP。

Spring不用创建InvocationHandler，只需要创建一个切面对象，将所有的非业务代码放在切面对象中完成即可。Spring会根据切面类以及目标类生成一个代理对象

aop:aspectj-autoproxy 让spring结合切面类与目标类自动生成代理对象

关于SpringIOC

发表于 2019-10-26 分类于 Spring相关

Spring的两大核心机制

  • 控制反转(IOC)/依赖注入(DI)

  • 面向切面编程(AOP)

如何使用IOC

  • 创建maven工程，pom.xml加入依赖
  • 创建实体类User
  • 传统的开发模式，手动new一个user对象

  • 利用IOC，在配置文件中添加需要管理的对象，格式xml，名字可以自定义

  • 从IOC中获取对象

配置文件

  • 通过配置bean标签来完成对象的管理

    • id: 对象名

    • class：对象的模版类。（所有交给IOC容器来管理的类必须由无参构造，因为spring底层是通过反射来创建对象的，调用的是无参构造）

  • 对象的成员变量通过property标签完成赋值

    • name：成员变量名
    • value: 成员变量值(基本数据类型，String可以直接赋值。如果是其他引用类型，不能通过value赋值)
    • ref: 将IOC另一个bean赋值给当前的成员变量(DI)

IOC底层原理

  • 读取配置文件，解析XML文件
  • 通过反射机制实例化配置文件中所配置的所有bean

Spring的工厂方法

IOC通过工厂模式创建bean的方法有两种

  • 静态工厂方法
  • 实例工厂方法

建国七十周年的一点思考

发表于 2019-10-01 更新于 2019-10-26 分类于 生活感想

弹指一挥间，中华人民共和国已经成立了七十年。
其实今天这太平时代当真来之不易，当年日本侵略者企图吞并中国，中华民族当真到了最危险的时候，东北告急，华北告急，上海南京武汉相继沦陷。我现在都有时候思考，在那样的局势下，如果我活在那个时代，会不会也去将生命挥洒在这片热土上，引刀成一快，不负少年头。
我们虽然不能假设，但是去可以做好现在。有时候我们会问，为什么要爱国呢？之前我也想过这个问题，但是现在我似乎有了答案。你什么要爱你妈妈呢，为什么要尊敬她，有能力了要为她分担压力呢？其实这是一个道理，你生活在祖国大地，一山一水养育了你，从你出生那天起国家就在尽可能为铺平道路，减免学杂费，废除了中国存在上千年的农业税，营造一个相对稳定和谐公平的社会。可以讲国家也像母亲一样生养了我们，这就是为什么要爱国的理由。
有时候其实我们对现在的社会的要求太高，希望她各方面都能和世界最强一较高下，但是毕竟新中国也才成立了七十周年，很多事情都需要一步一步的慢慢来，而且就效率而言，祖国已经表现的相当不错了，经济军事科技教育国际地位都可以说较1949年有质的飞跃。
梁启超先生曾言“少年智则国智,少年富则国富，少年强则国强”，其实现在来看，这话中的少年指的就是我们，我们正好处在这个节点，见证和参与中华民族伟大复兴。我们这代人是从小学到大学都受到国家的恩惠，不论世界形势如何变换，就像中华民族传统美德讲的”百善孝为先”一样，这个孝也同样适用于我们对待祖国。人生不过数十载，我们的前二三十年都是生活在这片土地上的，受其恩惠及哺育，所以爱国是应该存在我们血液基因里的东西，保持自己的思考，为自己的价值实现和国家民族的复兴奋斗终生。值此新中国成立七十周年之际，有所感慨，期待下个十年祖国与我们的改变。
