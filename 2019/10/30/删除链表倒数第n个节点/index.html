删除链表倒数第n个节点

发表于 2019-10-30 分类于 leetcode

给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

实例

解法1(直接) : 将链表长度求出，通过节点关系删除相应节点。

解法2(巧妙): 设两个指针，指在哑节点上。让第一个节点先移动至与第二个节点相隔两个节点处。再同时移动，至第一个节点为空。这时候，第二个节点正好在需要删除节点的前一个节点处。执行s.next = s.next.next;操作即可删除节点。
